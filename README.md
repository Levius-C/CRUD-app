# [CRUD app](https://project-react-app-831fc.web.app)

## Description

+ A CRUD application base on [firebase](https://firebase.google.com/)

+ Features

1. Log in with Google account

2. Create posts

3. Read posts from database

4. Like or unlike posts globally
